# 题型策略

仅在需要细分回答结构时读取本文件。

## 概念解释

- 先回答“它是什么”。
- 再补“作用 / 特点 / 使用语境”。
- 优先使用定义句、作用句、授权句。

输出模式：

```md
回答：
<概念定义>

补充依据：
- <作用或特点>
- <边界或注意点>
```

## 章节总结

- 先用目录或章节命中判断范围。
- 再提炼“定义/范围 -> 核心过程 -> 关键要点”。
- 优先使用带“过程包括”“主要作用”“主要内容”的正文块。

输出模式：

```md
回答：
<一句章节概括>

核心过程/要点包括：<过程 1>、<过程 2>、<过程 3>...
```

## 比较题

- 优先分别找 A 和 B 的定义。
- 再找同时提到两者、或包含“有时被称为”“更广”“着眼于”的关系句。
- 如果教材没有直接并列对照，就用“分别定义 + 关系句”回答。

输出模式：

```md
回答：
- A：...
- B：...
- 两者关系或区别：...
```

## 题目回查教材

- 从题干抽 2 到 5 个关键词。
- 优先定位定义、原则、步骤、输入输出、适用场景。
- 最后说明对应章节或页码。

输出模式：

```md
回答：
<该题主要考查的教材知识点>

补充依据：
- <定义/原则/步骤>

引用页码：
- 第 X 页
```
